# RealMySQL

순서대로 내용을 정리하지 않고 책의 권장 순서대로 정리할 예정입니다.

```
6장_실행 계획			 7장_쿼리 작성 및 최적화
8장_확장 검색			 9장_사용자 정의 변수
11장_스토어드 루틴			 13장_프로그램 연동
14장_데이터 모델링			 15장_데이터 타입
16장_베스트 프랙티스
```
